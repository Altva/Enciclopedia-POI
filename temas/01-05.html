HTTP

  • Hypertext Transfer Protocol (HTTP)

    • Protocolo de aplicación mayormente usado en la actualidad, siendo la base de lo que conocemos como internet.
    • Su propósito es la adecuación del lenguaje HTML y otros documentos web a través del internet.
    • Una pàgina web esta compuesta de objetos. Estos objetos pueden ser archivos HTML, imagenes, Java Applet, audio, video, etc.

FTP

    • Dentro de cada paquete de información hay una cabecera que contiene información de la dirección
    • Esta dirección esta grabada en forma definitiva en la tarjeta de Red por el fabricante.

SMTP

  • Protocolo de red basado en texto, utilizado para el intercambio de mensajes de correo electrónico entre computadoras u otros dispositivos.
  • En el SMTP los clientes y el servidor cuentan con 2 componentes importantes:
  • User Agents: Prepara el mensaje y lo ‘empaqueta’ para su transmisión.
  • Mail Transfer Agent: Transfiere el correo a través de Internet.

  • Mail Gateway

    • Un gateway orientado a correo electrónico, está diseñado para conectar dos protocolos de correo diferentes.
    • Ejemplo: Dos sistemas de correo SMTP y POP3

  • Limitaciones

    • Solo usa formato NVT 7 bit ASCII.
    • No tiene mecanismos de autenticación.
    • Los mensajes no se encriptan.
    • Susceptible al mal uso (spam).
